## 목적

특정 지갑의 송금 내역(입금/출금)을 조회한다.

- 사용자가 자신의 송금 이력을 확인하고, 동결/반환 상태를 추적할 수 있도록 한다
- 입금(해당 지갑이 도착지인 송금)과 출금(해당 지갑이 출발지인 송금)을 함께 조회한다

## 조회 항목

- 송금 식별자와 입금/출금 구분
- 코인 정보: 코인 식별자와 코인 심볼
- 송금 경로: 체인, 도착 주소, 도착 태그
- 금액 정보: 송금 수량, 수수료(입금은 0)
- 상태 정보: 송금 상태, 실패 사유, 동결 만료 시각
- 시각 정보: 생성 시각, 완료(성공/반환) 시각
- 커서 페이지 정보: 다음 페이지 커서와 다음 페이지 존재 여부

## 조회 대상

- 해당 지갑이 출발 지갑이거나 도착 지갑인 송금을 조회한다
- 구분 필터로 입금/출금을 가른다
  - 전체: 입금 + 출금 전체
  - 입금: 해당 지갑이 도착 지갑인 송금
  - 출금: 해당 지갑이 출발 지갑인 송금

## 지갑 소유권 검증

- 요청자가 해당 지갑의 소유자인지 검증한다
