## 목적

거래소별 포트폴리오의 보유 코인·잔고·평균매수가·현재가를 조회한다.

- 사용자가 특정 거래소의 투자 현황을 한눈에 파악할 수 있도록 데이터를 제공한다

## 용어 정의

- 사용자: 조회를 요청한 사용자
- 지갑: 조회할 거래소별 지갑

## 조회 항목

- 거래소 ID와 기축통화 심볼·총 잔고(가용 잔고 + 동결 잔고)
- 보유 코인 목록 — 코인별 심볼·이름·보유수량·평균매수가·현재가

## 검증

| 항목 | 규칙 |
|------|------|
| 지갑 존재 | 요청한 지갑이 존재해야 한다 |
| 지갑 소유권 | 지갑 소유자가 요청자 본인인지 검증한다 |
