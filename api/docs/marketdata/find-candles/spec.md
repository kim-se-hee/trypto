## 목적

캔들(OHLC) 차트 데이터를 조회한다. 시계열 캔들 데이터를 커서 기반 페이징으로 제공한다.

- 거래소별, 코인별 캔들 차트를 제공하기 위한 기능이다

### 사용처

- 마켓 탭에서 거래소·코인을 선택하면 캔들 차트를 표시한다

## 조회 항목

- 거래소·코인별 캔들(OHLC) 시계열 데이터: 각 캔들의 시가·고가·저가·종가와 시각
- 6개 주기(1분, 1시간, 4시간, 일, 주, 월)의 캔들
- 커서 기반 페이징으로 과거 캔들을 스크롤하며 조회
- 최신 윈도우 조회 시 아직 닫히지 않은 현재 구간의 캔들은 실시간으로 집계해 조회 결과에 포함

## 일 경계

일·주·월·4시간 봉이 새로 시작하는 시각은 거래소마다 다르다. 각 주기가 언제 시작하는지 정리하면(한국시간 기준):

| 주기 | 업비트·바이낸스 | 빗썸 |
|------|----------------|------|
| 일 | 매일 09시 | 매일 0시 |
| 주 | 월요일 09시 | 월요일 0시 |
| 월 | 1일 09시 | 1일 0시 |
| 4시간 | 09·13·17·21·01·05시 | 0·4·8·12·16·20시 |

업비트·바이낸스는 UTC 자정(= 한국 09시), 빗썸은 한국시간 자정(= UTC 15시)이 기준이라 모든 주기가 9시간 차이 난다.

## 검증

| 항목 | 규칙 |
|------|------|
| 주기 | `1m`, `1h`, `4h`, `1d`, `1w`, `1M` 중 하나 |
| 조회 개수 | 1 이상 200 이하. 미지정 시 기본값 60 |

## 정렬

- 시간 오름차순(과거부터)으로 반환한다

## 빈 결과

- 해당 조건의 캔들이 없으면 빈 결과를 반환한다

## 커서

- 커서가 없으면 최신 캔들부터, 값이 있으면 해당 시각 이전 캔들을 조회한다
