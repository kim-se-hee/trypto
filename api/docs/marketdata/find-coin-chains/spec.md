## 목적

특정 거래소의 특정 코인이 지원하는 체인(네트워크) 목록을 조회한다.

### 사용처

- **출금 화면**: 코인과 거래소를 선택하면 사용 가능한 네트워크 목록을 보여준다
- **입금 화면**: 입금 주소를 조회하기 전에 선택 가능한 네트워크 목록을 보여준다

## 조회 항목

- 코인이 지원하는 체인(네트워크) 목록
- 체인별 태그(메모) 입력 필요 여부

## 조회 대상

- 해당 거래소에 상장된 코인이 지원하는 체인(네트워크) 목록을 반환한다.
- 거래소에 상장되지 않은 코인을 조회하면 조회 대상이 존재하지 않는 것으로 처리한다.

## 응답 데이터

- 각 체인은 체인 식별자, 체인 이름, 태그(메모) 입력 필요 여부를 함께 전달한다.
- 태그(메모) 입력 필요 여부는 태그가 필수인 체인을 알려준다. 태그가 필수인 코인에 태그를 누락하면 자금을 잃는다.
