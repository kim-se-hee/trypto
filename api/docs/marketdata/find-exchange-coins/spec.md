## 목적

거래소에 상장된 코인 전체 목록을 조회한다.

- 거래소별 상장 코인 목록을 제공하기 위한 기능이다
- 여러 화면이 공통으로 참조하는 정적 데이터다

### 사용처

- **마켓 탭**: 거래소별 전체 코인 시세 목록을 표시한다
- **주문 화면**: 코인 심볼을 거래소 내 코인으로 매핑하는 데 쓴다
- **입출금 탭**: 거래소 전체 코인을 표시하고, 잔고와 코인 단위로 매핑하여 보유 여부를 판별한다

## 조회 항목

- 거래소에 상장된 코인 전체 목록
- 각 코인의 식별 정보(거래소 내 코인 식별자, 코인 식별자), 심볼, 이름

## 조회 대상

- 거래소가 존재해야 한다. 존재하지 않는 거래소로 조회하면 거래소 없음으로 처리한다.
- 해당 거래소에 상장된 코인 전체를 반환한다. 항목마다 거래소 내 코인 식별자, 코인 식별자, 심볼, 이름을 함께 제공한다.
