## 목적

거래소별 실시간 티커(시세)를 클라이언트에 실시간으로 전달한다.

- 마켓 탭, 포트폴리오 탭, 입출금 탭 등 실시간 시세가 필요한 모든 화면이 이를 구독해 현재가를 실시간으로 표시한다.

## 전달 항목

- 거래소별 실시간 티커: 현재가, 등락률, 거래대금
- 각 화면은 이 중 필요한 항목만 선택적으로 사용한다

## 최신값 묶음 전달

- 짧은 시간 창(약 50ms) 안에 들어온 같은 거래소·코인의 시세는 최신 1건만 남겨 거래소 단위로 묶어 한 번에 전달한다. 과도한 갱신을 막기 위함이다.

## 구독 단위

- 거래소 단위로 구독한다. 클라이언트는 현재 보고 있는 거래소 하나만 구독한다.
- 거래소 탭을 전환하면 기존 구독을 해제하고 새 거래소를 구독한다.

## 식별자 변환

- 전달 전에 거래소 시세의 외부 식별자(거래소·심볼)를 내부 코인으로 변환해 내려준다.
