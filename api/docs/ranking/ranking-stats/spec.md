## 목적

기간별 랭킹 통계를 조회한다.

## 용어 정의

- 기간: 일간/주간/월간 중 선택. 실행 주기가 아니라 수익률 산출 기간을 의미한다.
- 기준 날짜: 통계를 집계할 기준이 되는 날짜. 내부적으로 최신 날짜를 자동 결정한다.

## 조회 항목

- 참여자 수
- 최고 수익률
- 평균 수익률

## 입력 정보

- 기간(`period`): 일간/주간/월간 중 선택

## 기간 검증

| 항목 | 규칙 |
|------|------|
| period | `DAILY`, `WEEKLY`, `MONTHLY` 중 하나여야 한다 |

## 집계 규칙

- 선택한 기간의 최신 기준 날짜 랭킹을 집계한다.
- 참여자 수, 최고 수익률, 평균 수익률을 계산한다.
- 해당 기간의 랭킹 집계 결과가 없으면 조회할 수 없다.
- 기준 날짜는 내부적으로 최신 날짜를 자동 결정한다.

## 권한

- 인증 불필요 (공개 통계)
