## 목적

로그인 유저 본인의 랭킹 순위를 조회한다.

## 용어 정의

- 기간(`period`): 일간/주간/월간 중 선택
- 사용자: 랭킹을 조회할 사용자

## 조회 항목

- 본인의 순위(rank)
- 닉네임
- 수익률
- 거래 횟수
- 미참여 시 데이터 없음(`data: null`)

## 기간 검증

| 항목 | 규칙 |
|------|------|
| period | `DAILY`, `WEEKLY`, `MONTHLY` 중 하나여야 한다 |

## 조회 처리 규칙

1. 조회할 사용자를 입력받는다
2. 랭킹 집계 결과에서 해당 사용자·기간의 최신 기준 날짜 순위를 조회한다
3. 랭킹 데이터가 있으면 순위, 닉네임, 수익률, 거래 횟수를 반환한다
4. 미참여(데이터 없음) 시 `data: null`을 반환한다 (에러가 아님)

## 미참여 처리

- 랭킹에 참여하지 않은 사용자를 조회해도 에러가 아니라 정상 응답으로 처리하며, 순위 없음을 반환한다.
