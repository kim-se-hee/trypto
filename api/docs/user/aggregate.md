User : 회원. 소셜 신원을 직접 갖지 않고 SocialAccount 를 ID로만 참조한다. 탈퇴 시각(deletedAt)을 가지며, 재가입 제한 기간(30일) 검증을 도메인 규칙으로 소유한다.
SocialAccount : 소셜 신원(제공자·제공자 회원번호)의 소유자. 현재 연결된 회원을 ID로만 참조하는 독립 애그리거트로, 회원 연결·해제와 연결 여부 판별을 스스로 책임진다. 소셜 신원 1개는 동시에 회원 1명과만 연결된다.
Feedback : 회원이 운영자에게 보낸 사용 후기·개선 요청. 작성자를 ID로만 참조하는 독립 애그리거트로, 회원과 생명주기가 분리되어 작성자가 탈퇴해도 보존된다. 본문 길이 규칙(20자 이상 1000자 이하, 공백만인 본문은 빈 본문으로 간주)을 도메인 규칙으로 소유한다.
