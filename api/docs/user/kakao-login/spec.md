## 목적

카카오 계정으로 서비스에 로그인한다. 아직 회원이 아니면 로그인 과정에서 자동으로 회원가입까지 처리한다. 로그인에 성공하면 일정 기간 로그인 상태가 유지되어, 그 동안 재인증 없이 서비스를 이용할 수 있다.

## 용어 정의

- 소셜 신원 = "어느 소셜 제공자(카카오·구글)의 몇 번 회원인지"를 나타내는 정보. 제공자 이름과 그 제공자가 매긴 회원번호를 함께 가진다. (예: 카카오의 1234567번)
  - 제공자가 다르면 회원번호가 우연히 같아도 다른 사람이다. 그래서 제공자와 회원번호를 함께 봐야 회원을 특정할 수 있다.
  - 우리 서비스 회원 한 명은 소셜 신원을 하나만 가진다(1:1).

## 회원 확인 및 등록

- 카카오 인증을 통과한 사용자의 소셜 신원(제공자는 카카오, 회원번호는 카카오 회원번호)으로 이미 가입한 회원이 있는지 찾는다.
- 이미 가입한 회원이 있으면 그 회원으로 로그인한다.
- 가입한 회원이 없으면 우리 서비스 회원을 새로 만들어 자동 회원가입하고 그 소셜 신원이 방금 만든 회원을 가리키도록 기록한 뒤 로그인한다.
- 같은 사람이 카카오·구글로 각각 로그인하면 서로 다른 회원이 되며 한 사람이 두개의 아이디를 갖게 된다. 하나로 합치지 않는다.

## 신규 회원 생성

- 자동 회원가입으로 만들어지는 회원에게는 서비스가 유일한 닉네임을 자동으로 부여한다. 사용자는 이후 닉네임을 변경할 수 있다.
- 신규 회원의 포트폴리오는 공개 상태로 시작한다.

## 개인정보 수집 범위

- 카카오 로그인 시 회원 식별에 필요한 카카오 회원번호만 수집한다.
- 이메일을 포함한 그 밖의 개인정보는 동의 요구·수집·저장하지 않는다.

## 세션 유지

- 로그인에 성공하면 로그인 세션이 생성된다.
- 세션은 발급 후 일정 기간 유효하며, 유효한 동안 재인증 없이 로그인 상태가 유지된다.
- 세션 유효 기간이 지나면 로그인 상태가 해제되며, 다시 이용하려면 로그인을 새로 해야 한다.

## 로그인 실패

- 카카오 인증에 실패하거나(인가 코드가 유효하지 않거나 만료된 경우 등), 카카오로부터 회원 식별 정보를 얻지 못하면 로그인은 실패한다.
- 로그인이 실패하면 회원 생성·세션 발급 등 어떤 변화도 남기지 않는다.
