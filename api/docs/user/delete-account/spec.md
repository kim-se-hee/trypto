## 목적

회원이 서비스 이용을 그만두고 계정을 탈퇴한다. 탈퇴하면 로그인과 서비스 이용이 즉시 중단된다. 이미 쌓인 투자 기록과 랭킹·공개 포트폴리오 노출은 사라지지 않고 그대로 유지된다.
감사·통계를 위해 데이터 자체는 지우지 않고 탈퇴 상태로 보존한다.

## 용어 정의

- 활성 회원 = 아직 탈퇴하지 않은 회원.
- 탈퇴 회원 = 탈퇴 처리된 회원. 더 이상 로그인할 수 없지만 데이터와 공개 노출은 유지된다.

## 권한

- 본인만 자신의 계정을 탈퇴할 수 있다. 로그인 세션의 주인에 한한다.

## 탈퇴 처리

- 회원은 활성 → 탈퇴 상태로 전이한다.
- 탈퇴 회원은 다시 활성 상태로 되돌아가지 않는다. 재이용은 오직 신규 가입으로만 가능하며 계정 복구·재활성화는 없다.
- 이미 탈퇴한 회원은 다시 탈퇴할 수 없다.
- 탈퇴는 진행 중인 활동 유무와 관계없이 즉시 처리된다. 별도의 대기·유예 상태를 두지 않는다.
- 탈퇴 시 회원의 닉네임은 서비스가 자동 부여하는 익명 닉네임(예: 탈퇴한사용자123)으로 바뀐다. 원래 닉네임은 즉시 해제되어 다른 회원이 사용할 수 있다.

## 데이터 보존

- 탈퇴해도 회원 정보와 지갑·잔고·거래내역·포트폴리오·후회분석 등 연관 데이터는 지우지 않고 보존한다.

## 노출 유지

- 탈퇴해도 해당 회원과 그 투자 기록은 수익률 랭킹에 그대로 남는다.
- 공개 포트폴리오 열람도 탈퇴 전과 동일하게 유지된다. 포트폴리오 공개 여부는 탈퇴 시점의 설정을 그대로 따른다.
- 단, 노출되는 이름은 원래 닉네임이 아니라 탈퇴 시 부여된 익명 닉네임이다. 보는 사람은 이 이름으로 해당 랭커가 탈퇴했음을 알 수 있다.

## 부수 효과

- 탈퇴 시 해당 회원의 모든 로그인 세션이 종료되어 즉시 로그아웃된다.
- 탈퇴 시 해당 회원의 미체결 주문(아직 체결되지 않은 지정가 주문 등)은 모두 취소된다.
