## 목적

사용자가 지금까지 시작한 라운드의 누적 횟수를 조회한다.

- 활성 라운드 조회는 진행 중인 라운드가 없으면 조회할 수 없으므로, 라운드를 한 번도 시작한 적 없는 신규 사용자와 라운드를 종료한 기존 사용자를 구분할 수 없다.
- 두 상태를 구분해야 신규 사용자에게는 라운드 생성을 먼저 요구하고, 기존 사용자에게는 라운드 없이도 화면 열람을 허용할 수 있다.

## 조회 항목

- 누적 라운드 횟수

## 누적 라운드 횟수 조회

- 사용자가 시작한 전체 라운드의 개수를 제공한다.
- 라운드를 한 번도 시작하지 않았으면 0을 제공한다.
- 종료되거나 파산한 라운드도 누적 횟수에 포함한다.
